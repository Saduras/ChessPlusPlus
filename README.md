# ChessPlusPlus

A simple chess game including it's own chess engine and AI.

## Features
- Simple window view with drag'n'drop interaction
- Move generation
- Move validation
- Agents:
  - Human player
  - Random move AI
  - MiniMax AI with alpha-beta-pruning

## Develop environment
- Windows 10
- Visual Studio 2017

## Dependencies
- [SFML](https://www.sfml-dev.org/)
